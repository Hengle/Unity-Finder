#### Install

Find the manifest.json file in the Packages folder of your project and edit it to look like this:
``` json
{
  "dependencies": {
    "com.litefeel.com.litefeel.finder": "https://github.com/litefeel/com.litefeel.finder.git",
    ...
  }
}
```
